# Student-Study-Habits-and-Stress-Analysis
Excel-based project analyzing student data
This project analyzes student performance data using Excel. 
It includes data cleaning, calculations, charts, and insights about students' performance trends.

## Objectives
- Analyze overall student performance.
- Identify students at risk of failing or underperforming.
- Visualize data using charts for better understanding.
- Generate insights for teachers or academic planners.

## Analysis Performed
1. **Data Cleaning**: Removed duplicates, corrected errors in marks.
2. **Formulas Used**: SUM, AVERAGE, IF, etc.
3. **Charts**: Bar charts, pie charts to visualize study hours and stress level.

## Future Work
- Automate analysis using  Power Query.
- Add more datasets (like extracurricular activities).
- Predict future student performance using machine learning.
